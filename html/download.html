This release of Pippy requires PalmOS 3.5 on a device with at least 4 M of RAM (This maximizes the dynamic heap size, which is proportional to the amount of RAM for PalmOS 3.5.). Before downloading, please read the license agreement.

Pippy Palm Install Files Only (.prc files)

To obtain the latest release of Pippy (pre-built .prc files) to install on your Palm, download the prc files using one of the formats below. This will provide you with two files that you install directly on the Palm device: pylib.prc and pippy.prc.
  • pippy_0_6beta.zip [124 K]
  • pippy_0_6beta.tar.gz [123 K]
  • pippy_0_6beta.sit [110 K]
  • pippy_0_6beta.sit.hqx [150 K]
    •  The installation and usage instructions are found in README.interactive.

    Pippy Source Distribution

    The source distribution pippy-0.6beta-src contains the source files (modified for Python-1.5.2+) and the interactive interpreter application.
  • pippy-0.6beta-src.tar.gz [2137 K]
    •  Please be sure to read the README, TOOLS and BUILD files found in the docs/ directory when you unpack the distribution. These files provide instructions for obtaining the necessary tools and building Pippy.

    Pippy CVS Repository

    For access to the latest changes, visit the Pippy CVS repository.

    To login, use the following statement and hit <enter> when prompted for a password: 
        cvs -d:pserver:anonymous@endeavors.com:/cvsroot login
    To checkout the sources: 
        cvs -z3 -d:pserver:anonymous@endeavors.com:/cvsroot co pippy
