Pippy - Mailing Lists

We have created several mailings lists for the Pippy Project. These are listed below.

How-To Subscribe

To be added to one of the Pippy mailing lists, simply send a message to majordomo (our list manager).

There are two ways to do it: 
address--    To: ListName-request@endeavors.com
message--    subscribe

 OR

address--    To: majordomo@endeavors.com
message--    subscribe ListName
Majordomo understands several commands, and is not limited to a single command per message (it will process commands until reaching end-of-message or the command "end"). The command "help" will tell you about all the other commands. The commands which Majordomo understands are: 
    subscribe <list> [<address>]
    unsubscribe <list> [<address>]
    which [<address>]
    who <list>
    info <list>
    index <list>
    get <list>
    lists
    help
    end
You can get detailed explanations of all of these by asking for "help" from Majordomo (send a message containing just the word "help" as the message text to majordomo@endeavors.com).
